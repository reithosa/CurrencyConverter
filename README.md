# Currency Converter - Конвертер валю
---

Мобильное приложение на Android, использующие API Open Exchange Rate для получения актуальных курсов валют относительно доллара.

### Использование приложения
Для работы приложения необходимо получить собственный API-ключ в [Open Exchange Rate](https://openexchangerates.org/) и вставить его в поле ввода на начальном экране. Сохранение API-ключа не реализовано, так что необходимо вводить API-ключ при каждом использовании приложения. 

<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/login.jpg" alt="load">
</div>
<center><small>Окно ввода API-ключа</small></center>

Если введённый ключ корректный, то приложение отобразит следующие окно:

<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/load.jpg" alt="load">
</div>
<center><small>Окно загрузки</small></center>

Не забудьте установить устойчивое интернет-соединение, иначе приложение не сможет загрузить курсы валют. Нажитием на валюту вы можете смотреть её стоимость в других валютах. В случае успешной загрузки курсов валют отобразится следующие окно:

<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/result.jpg" alt="result">
</div>
<center><small>Главное окно приложения</small></center>

Вы можете изменить отображаемые валюты на главном экране нажав на кнопку в левом верхнем углу (стандартно выбраны все валюты):
