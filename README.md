# Currency Converter - Конвертер валю
---
Мобильное приложение на Android, использующие API Open Exchange Rate для получения актуальных курсов валют относительно доллара.
<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/main.jpg" alt="main">
</div>
---
### Использование приложения
Для работы приложения необходимо получить собственный API-ключ в [Open Exchange Rate](https://openexchangerates.org/) и вставить его в поле ввода на начальном экране. Сохранение API-ключа не реализовано, так что необходимо вводить API-ключ при каждом использовании приложения. Если введённый ключ корректный, то приложение отобразит следующие окно:
<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/load.jpg" alt="load">
</div>
Не забудьте установить устойчивое интернет-соединение, иначе приложение не сможет загрузить курсы валют. В случае успешной загрузки курсов валют отобразится следующие окно:
<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/result.jpg" alt="result">
</div>
Вы можете изменить отображаемые валюты на главном экране нажав на кнопку в левом верхнем углу:
<div align="center">
    <img src="https://raw.githubusercontent.com/reithosa/CurrencyConverter/main/screenshots/change.jpg" alt="change">
</div>
